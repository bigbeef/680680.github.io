redis for windows 安装教程

发表于 2016-07-08 | 分类于 redis

前言

redis在linux操作系统下安装已经足够简单，官方网站也写的特别清楚，但是在我们开发中，经常需要在windows操作系统下启动redis服务，下面我们一起搭建一个windows操作系统的redis服务

1.下载redis for windows

下载地址：http://pan.baidu.com/s/1hrYynq8

2.解压redis

下载后解压，你会看到如下目录，下面是我的目录，记住你的绝对目录地址
（我的是：C:\Users\Administrator\Desktop\redis64-3.0.501）

3.运行redis

按键win+R，打卡cmd
依次输入：cd C:\Users\Administrator\Desktop\redis64-3.0.501
redis-server.exe redis.windows.conf

安装成功（这个界面千万不到关闭，否则redis也会关闭）

4.测试

我们测试一下：
同样打开cmd 输入telnet 127.0.0.1 6379，此时会打卡redis命令控制界面
我们依次输入：set test 123456
get test

如果你能弹出结果，恭喜你安装成功
